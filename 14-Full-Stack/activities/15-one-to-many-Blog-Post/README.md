# One-To-Many Blog Post

## `Summary`
A very simple application that depicts the use of `one-to-many` relationship. 

## `Model`

- User
- Blog
- Comment

## `Relationship`

**User** `hasMany` **Blog** 
> One `User` can write `many` blogs

**Blog** `hasMany` **Comment**
> One `Blog` can have many `Comments`

## `Technologies Used:` 

- Express
- Node
- MySQL 
- Bootstrap 
- Javascript 
- AJAX 
- Sequelize
